The main target of this project is Ubuntu 18, as the older version of

[GNU M4](https://www.gnu.org/software/m4/) is an implementation of the traditional Unix macro processor.

The initial target of this project was Ubuntu 18, whose older version
of this utility failed some builds.

**GNU m4** is an open source project hosted on
[Savannah](https://savannah.gnu.org/projects/m4/).
